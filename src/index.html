Python Učilica koristi Pyodide (Python za browser), koji zahtijeva da stranica bude poslužena preko web servera — ne radi ako se HTML otvori dvoklikom (file://).

Pokreni mini server u folderu gdje je index.html:

python -m http.server 8000

Zatim otvori u browseru:

http://localhost:8000

Ovo je sigurnosno ograničenje browsera (CORS), ne greška u aplikaciji. Electron desktop verzija ovo automatski rješava.
